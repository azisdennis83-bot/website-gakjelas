# 📘 CBT Ujian Sekolah

Aplikasi **Computer Based Test (CBT)** sederhana berbasis HTML, CSS, dan JavaScript.  
Mendukung login peserta, soal pilihan ganda, timer, hasil ujian, serta rekap nilai.

---

## 🚀 Cara Pakai

1. **Clone / download repo ini**  
   ```bash
   git clone https://github.com/username/cbt-ujian.git
   ```

2. **Buka file `index.html`** di browser (Chrome/Edge).  
   - Bisa digunakan langsung secara offline.  
   - Atau online via **GitHub Pages**.

---

## 🌐 Deploy ke GitHub Pages

1. Buka GitHub → buat repository baru dengan nama `cbt-ujian`.  
2. Upload file `index.html` ke repository.  
3. Masuk ke menu **Settings → Pages**.  
4. Pada bagian **Source**, pilih:
   - Branch: `main`
   - Folder: `/root`  
5. Klik **Save**, tunggu ±1 menit.  
6. Website CBT kamu bisa diakses di:
   ```
   https://username.github.io/cbt-ujian/
   ```

## ✨ Fitur Utama
- Login peserta ujian
- Soal pilihan ganda
- Timer otomatis (60 detik default)
- Hasil ujian per peserta (download CSV)
- Rekap nilai semua peserta (download CSV)
- Tampilan modern & responsif

---

## 📝 Catatan
- Soal dan akun peserta bisa diubah langsung di bagian JavaScript pada file `index.html`.  
- Tidak membutuhkan server/database, cukup file HTML.  
- Cocok untuk latihan CBT sederhana.
